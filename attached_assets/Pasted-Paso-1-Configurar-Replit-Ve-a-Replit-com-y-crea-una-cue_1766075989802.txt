Paso 1: Configurar Replit
Ve a Replit.com y crea una cuenta (si no tienes).

Haz clic en el botón azul "+ Create Repl".

En "Template", busca y selecciona "FastAPI" (o simplemente Python, pero FastAPI ya trae estructura web).

Ponle de título: ProyectoXYZ-MVP.

Dale a "Create Repl".

Replit instalará paquetes automáticamente, pero vamos a definir qué necesitamos para que el servidor web funcione.

Paso 3: El Backend (El "Cerebro" Temporal)
Vamos a configurar un servidor simple que sirva tu diseño y tenga un endpoint para responder mensajes. Por ahora, el "cerebro" solo hará eco, luego le conectaremos la IA real.

Abre (o crea) el archivo main.py y pega este código:

Python

from fastapi import FastAPI, Request
from fastapi.responses import HTMLResponse
from fastapi.staticfiles import StaticFiles
from fastapi.templating import Jinja2Templates
import uvicorn

app = FastAPI()

# Configuración de templates y archivos estáticos
templates = Jinja2Templates(directory="templates")

# Endpoint para servir la Interfaz de Usuario (UI)
@app.get("/", response_class=HTMLResponse)
async def read_root(request: Request):
    return templates.TemplateResponse("index.html", {"request": request})

# Endpoint del Chat (Aquí luego conectaremos tu LLM real)
@app.post("/chat")
async def chat_endpoint(payload: dict):
    user_message = payload.get("message")
    
    # --- AQUÍ IRA LA LÓGICA DE TU IA MÁS ADELANTE ---
    # Por ahora, simulamos una respuesta empática simple
    bot_response = f"Entiendo que dices: '{user_message}'. Cuéntame más sobre cómo te sientes."
    
    return {"response": bot_response}

# Para correrlo en Replit
if __name__ == "__main__":
    uvicorn.run("main:app", host="0.0.0.0", port=8080, reload=True)

    <script>
        const chatContainer = document.getElementById('chat-container');
        const userInput = document.getElementById('user-input');
        const sendBtn = document.getElementById('send-btn');

        // Función para agregar mensaje al chat
        function appendMessage(text, isUser) {
            const wrapper = document.createElement('div');
            wrapper.className = "flex gap-4 max-w-3xl mx-auto " + (isUser ? "justify-end" : "");
            
            const avatar = isUser ? '' : `<div class="w-8 h-8 rounded-full bg-gradient-to-br from-purple-400 to-rose-400 flex-shrink-0"></div>`;
            const bubbleClass = isUser ? "bg-gray-100 px-4 py-2 rounded-2xl rounded-tr-sm" : "text-gray-800 leading-relaxed";
            
            wrapper.innerHTML = `
                ${!isUser ? avatar : ''}
                <div class="${bubbleClass}">
                    ${text}
                </div>
            `;
            chatContainer.appendChild(wrapper);
            chatContainer.scrollTop = chatContainer.scrollHeight;
        }

        // Función para enviar
        async function sendMessage() {
            const text = userInput.value.trim();
            if (!text) return;

            // 1. Mostrar mensaje del usuario
            appendMessage(text, true);
            userInput.value = '';

            // 2. Enviar al backend
            try {
                const response = await fetch('/chat', {
                    method: 'POST',
                    headers: { 'Content-Type': 'application/json' },
                    body: JSON.stringify({ message: text })
                });
                const data = await response.json();
                
                // 3. Mostrar respuesta de la IA
                appendMessage(data.response, false);
            } catch (error) {
                console.error("Error:", error);
                appendMessage("Lo siento, tuve un error de conexión.", false);
            }
        }

        // Event Listeners
        sendBtn.addEventListener('click', sendMessage);
        userInput.addEventListener('keypress', (e) => {
            if (e.key === 'Enter' && !e.shiftKey) {
                e.preventDefault();
                sendMessage();
            }
        });
    </script>

Paso 5: Desplegar y Probar
En la parte superior de Replit, verás un botón verde grande que dice "Run". Haz clic.

Se abrirá una consola a la derecha instalando las librerías.

Una vez termine, verás aparecer una ventana de "Webview" dentro de Replit con tu diseño.

Tip Pro: Haz clic en el ícono de "abrir en nueva pestaña" (New Tab) en la esquina de la ventanita del navegador de Replit para verlo en pantalla completa y apreciar el diseño responsive.
